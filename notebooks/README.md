# Notebooks Directory

Jupyter notebooks organized by learning day.

## Structure

```
notebooks/
├── day1_foundations/      # Theory, SAR physics, preprocessing
├── day2_implementation/   # Model building and training
├── day3_evaluation/       # Evaluation and analysis
├── day4_improvements/     # Advanced architectures
├── day5_compression/      # Rate-distortion analysis
├── day6_inference/        # Pipeline development
└── day7_finalization/     # Documentation and packaging
```

## Day 1: Foundations (8+ hours)
- `01_autoencoder_theory.ipynb` - Information bottleneck, PCA connection
- `02_sar_physics.ipynb` - Backscatter, speckle, geometry
- `03_loss_functions.ipynb` - MSE, SSIM implementation
- `04_preprocessing.ipynb` - Complete preprocessing pipeline
- `05_day1_exercise.ipynb` - Comprehensive Day 1 exercise

## Day 2: Implementation (6 hours)
- `01_encoder_decoder.ipynb` - Building encoder/decoder
- `02_autoencoder.ipynb` - Complete autoencoder
- `03_training.ipynb` - Training infrastructure
- `04_day2_exercise.ipynb` - Train first model

## Day 3: Evaluation (4 hours)
- `01_metrics.ipynb` - Implementing quality metrics
- `02_failure_analysis.ipynb` - Finding and analyzing failures
- `03_visualization.ipynb` - Creating visualizations
- `04_day3_exercise.ipynb` - Comprehensive evaluation

## Day 4: Improvements (4 hours)
- `01_residual_blocks.ipynb` - Residual connections
- `02_unet_style.ipynb` - Skip connections
- `03_attention.ipynb` - Attention mechanisms
- `04_architecture_comparison.ipynb` - Comparing architectures

## Day 5: Compression (4 hours)
- `01_rate_distortion.ipynb` - Rate-distortion analysis
- `02_bottleneck_sweep.ipynb` - Compression ratio experiments
- `03_codec_comparison.ipynb` - Comparison with JPEG2000

## Day 6: Inference (4 hours)
- `01_full_image_pipeline.ipynb` - Processing full images
- `02_patch_blending.ipynb` - Seamless reconstruction
- `03_geotiff_io.ipynb` - Reading/writing GeoTIFF

## Day 7: Finalization (4 hours)
- `01_final_evaluation.ipynb` - Final model evaluation
- `02_documentation.ipynb` - Generating documentation
- `03_deployment.ipynb` - Creating deployment package
